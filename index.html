Wine review, price and selection

When you go to whole foods, which wine would you choose?

Wine is made virtually every country in the world. There are historic regions such as France,Italy and Germany. There are also some new regions such as US,Australia and Chile with hotter climates.

The wine markets always seperate wines based on countries, it's hard to find a decent wine for a reasonable price, plus the promotion will make this choice even harder.
At first glance, the Wines' labeling are always confusing to those who doesn't know any wine but invited to a party, you also don't wanna throw more money but ended up with a bad one.


What about this visualization?

This visualization will help you decide the origin of the wine. The original data contains reviews and price. I also introduce a new factor called points per dollar to measure the qulity.



In this visualization, the x axis shows the average price for a country's wine, the y axis shows the average reviews(total 100).

This size of the circle is based on the review per price(points per dollar). This is an important factor for measuring wine quality.

When mouseover the circles, the points per dollar factor will show along with the country name. The similar color represents similar value of the factor. Interestingly, the 'old' regions like France, Italy, Germany show very high points per dollar ratio while Chile,Argentina,Greece deliver lower ppd product.

When click on the legend, the specific country data will be filtered out; The data will be reset if click the same country again.





About this visualization

Scenes

This project was created as slides using FullPage.js library.The first scene explains what to expect for this visualization. The second scene introduces how to use the graph. This method can guarantee visual consistency.

Annotations

The annotation is represented by the legend at the bottom of the graph. The graph also has tooltips when hovering over the data points. The tooltips will disappear when mouseout.

Parameters

In the graph, the country is the parameter for filtering data, the circle is another one for different color as similar color represents similar size of circle.

Triggers

The graph change will be triggered when users click on the legend. The graph can be reset if the user click sagain on the same country. The slide change can be triggered when click on the navigation bar in the first scene.
